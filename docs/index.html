System Overview

SprayCraft is an Agriculture Cyber-Physical System (A-CPS) that combines graph-based spatial analysis with TSP route optimization to direct spraying drones to apply the right pesticide dosage, at the right place, via the shortest possible path.

Fig. 1 — SprayCraft end-to-end system: from disease image acquisition to GPS-ready drone spray waypoints.

Fig. 2 — SprayCraft method block diagram: Diseased Locations → Spatial Analysis → Hotspot Detection → Intensity Calculation → Path Computation.

🌱
Plant diseases cause up to 20% of global produce loss. SprayCraft enables drones to spray smarter — concentrating pesticide on hotspot clusters while reducing waste in low-risk areas.

The Problem

Disease spread in farmland is spatially heterogeneous and governed by the disease triangle: the interaction of environment, host plant, and pathogen. Conventional uniform spraying ignores this entirely.

Disease Triangle — Environment × Host × Pathogen drives spatial clustering.

Disease propagation models: Logistic, Slow Rise, and Exponential growth.

Multiple disease hotspots in farmland — infection clusters vary in intensity.

Benefits of variable rate precision spraying over uniform application.

❌ Uniform Spraying

Fixed dosage everywhere — wastes pesticide in low-risk zones, under-treats hotspots, promotes resistance, harms beneficial insects.

✅ SprayCraft

Hotspot probability drives dosage — more pesticide where infection clusters, less where risk is low. Near-optimal drone routing minimizes travel.

Normal path vs. SprayCraft near-optimal tour — significant reduction in travel distance.

Full boustrophedon coverage pattern ensures zero gaps in spray application.

SprayCraft Method

A unified two-stage pipeline: spatial graph analysis for hotspot scoring, then TSP + boustrophedon path generation for drone routing.

1

Graph Construction

Each diseased instance becomes a node with feature = diseased area. Edges connect nodes within pathogen dispersal radius, weighted by inverse distance.

2

Message Passing (Hotspot Detection)

Iterative neighbor aggregation: hu(k+1) = UPDATE(hu(k), AGG({hv(k) · 1/W(u,v) : v ∈ N(u)})). Nodes with highest aggregated features = hotspots.

3

Hotspot Scoring

Normalize node features → hotspot probability score pu ∈ [0,1]. Directly drives spray dosage.

4

TSP Route (Christofides)

MST + minimum-weight matching on odd-degree nodes → Eulerian → Hamiltonian circuit. Provably within 1.5× of optimal.

5

Boustrophedon Coverage Paths

Serpentine parallel sweeps at each diseased location. Path spacing = 2× spray radius (from hotspot score). 100% area coverage guaranteed.

6

GPS Waypoint Export

Haversine formula converts meter-based coordinates to GPS lat/long. Output is drone-ready flight plan.

3a. Spatial Graph Representation

Graph G = (V, E, F): nodes = disease instances, edges = proximity within dispersal radius.

Graph representation of field: node color encodes diseased area (feature value).

36 diseased instances distributed across 250×250 m synthetic farmland.

Spatial layout of all diseased locations — basis for graph construction.

3b. Message Passing & Hotspot Detection

Message passing: each node aggregates infection signals from neighbors across iterations k and k+1.

Hotspot detection algorithm: data loading → graph build → message passing → normalization → scoring.

Hotspots identified after message passing: darker red = higher infection probability.

Hotspot probability heat map after two rounds of message passing. Nodes 14 & 15 emerge as primary hotspots.

3c. TSP Route Optimization (Christofides)

TSP formulation: Hamiltonian circuit visiting all diseased locations exactly once.

Computed near-optimal tour — cost: 1,056.13 m across 36 locations.

3d. Boustrophedon Coverage Paths

Boustrophedon (serpentine) path: alternating horizontal passes for full area coverage.

Path computation decision tree: variable rate vs. constant rate sprayer logic.

3e. Sprayer Adaptation

Conical spray pattern geometry: effective coverage area = π·R² adjusted by flight altitude.

✅ Variable Rate Sprayer (VRS)

PWM/PID flow control. Hotspot score modulates dosage in real time. Altitude stays constant.

spray_radius = 2 − 1 × hotspot_prob
flow_rate ∝ hotspot_probability
altitude = constant

⚙️ Constant Rate Sprayer

Fixed flow rate. Effective coverage area controlled by adjusting drone flight altitude per location.

Area = π × R²
altitude ↓ → R ↓ → denser spray
altitude ↑ → R ↑ → wider spray
